安装autojump
========================

ubuntu
------------------------

首先安装autojump软件包, ::

    sudo apt install autojump

在bashrc或者zshrc中添加 ::

    [ -f /usr/share/autojump/autojum.sh ] && source /usr/share/autojump/autojump.sh
